Sistema de software que permite la gestión y control del mantenimiento de sus clientes empresariales, así como la segmentación de estos por gustos, edades y profesiones ante las posibles demandas del mercado.

DIRECTOR DEL PROYECTO
Isabel Losilla Barrientos
PATROCINADOR DEL PROYECTO
Alejandro Bogantes Salazar
OBJETIVO GENERAL
Desarrollar un sistema de software que permita la gestión y control del mantenimiento de sus clientes empresariales, gustos y profesiones para la segmentación de estos ante las posibles demandas del mercado de acuerdo con sus gustos e intereses, utilizando la metodología de desarrollo de software ágil.
OBJETIVOS ESPECIFICOS
+ Identificar la situación actual con respecto a los clientes empresariales del mercado para la obtención de datos relevantes como gustos, intereses, profesiones para conformación de un almacén de datos que permita su adecuado control y mantenimiento ante las necesidades de las empresas.

+ Analizar la aplicación de software tomando en cuenta el estudio de los datos recolectados de los clientes empresariales para mejorar su adecuado control y mantenimiento ante las necesidades de las empresas.

+ Diseñar la aplicación de software tomando en cuenta el estudio y análisis de los datos recolectados de los clientes empresariales para mejorar su adecuado control y mantenimiento ante las necesidades de las empresas utilizando la metodología ágil y el modelado UML.

+ Desarrollar un prototipo funcional del sistema de acuerdo al estudio y análisis de los datos recolectados de los clientes empresariales para mejorar su adecuado control y mantenimiento ante las necesidades de las empresas utilizando el software Adobe XD
DESCRIPCIÓN DEL PROYECTO
+ El proyecto es una aplicación web llamada Sotri

+ Es una aplicación que permitirá el envío de correo electrónicos a clientes empresariales registrados en una base de datos.

+ Los usuarios podrán registrar los clientes, sus gustos y profesiones con el objetivo de poder segmentar los clientes por intereses o sectores.

+ En la aplicación se permitirá el envío de encuestas, promociones por medio del correo electrónico registrado de cada uno de los clientes.

+ Esta aplicación web será desarrollada con el lenguaje de programación de JavaScript haciendo uso de la librería de React.

+ Contará con un Dashboard en la página principal donde el cliente podrá ver y analizar el resultado de las encuestas enviadas con respuesta exitosa.
SUPUESTOS
+ Se asumen que el usuario cuenta con conexión a internet estable.

+ Se asume que el usuario posee conocimientos básicos en el manejo de un computador o un teléfono móvil.

+ Se asume que la aplicación para solucionar una necesidad en el mercado empresarial.

+ Se asume que emplearan las mejores practicas en el desarrollo de software para la creación y cumplimiento de todos los requerimientos solicitados a este proyecto.
LIMITACIONES
+ Conexión de internet inestables.

+ Poca experiencia en el análisis, diseño y desarrollo aplicaciones web.

+ Los miembros del proyecto poseen diferentes responsabilidades lo cual limita los tiempos disponibles para desarrollar el proyecto.

+ La documentación del sistema estará escrita en español.

+ Se cuenta con un equipo de 5 personas para desarrollar el proyecto completo.

Descripción del problema

Analizando una serie de productos actuales contra la necesidad del mercado detectamos que los mismos no siempre cumplen las expectativas de las empresas que los adquieren, ya sea por su nivel de complejidad de su uso o por la poca adaptabilidad con la que cuentan, lo cual nos brindó la idea de crear un producto que pudiera ser personalizado a criterio de la empresa.
